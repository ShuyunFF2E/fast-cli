## fast-cli

> Quickly create react, Vue or applications without front-end framework

## Install
```bash
# yarn
$ yarn global add shuyun-fast-cli

# npm
$ npm install -g shuyun-fast-cli
```

## Usege

```bash
$ fast-cli init

# install packages
$ cd my-app && npm install

# start
$ npm start
```
