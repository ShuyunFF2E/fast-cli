## fast-cli

> Quickly create react, Vue or applications without front-end framework

## Install
```bash
# yarn
$ yarn global add fast-cli

# npm
$ npm install -g fast-cli
```

## Usege

```bash
# Initialize new project
$ fast-cli init
```
